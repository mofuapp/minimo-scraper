# ミニモ サロンスクレイパー 💅

ミニモ（minimo）から「いいね」が5以下のサロンを自動収集するツールです。

## 機能

- 🔍 **自動スクレイピング**: ネイル、マツエク、眉毛サロン、エステ・リラクの各カテゴリを検索
- 📊 **フィルタリング**: いいね数が指定値以下のサロンのみを抽出
- 📋 **重複チェック**: 既にスプレッドシートにあるサロンは自動でスキップ
- 💾 **データ保存**: CSVファイルに自動保存、ダウンロード可能
- 🎨 **使いやすいGUI**: ボタン一つで操作可能

## セットアップ

### 1. Python環境の準備

Python 3.9以上が必要です。

```bash
cd minimo-scraper
```

### 2. 依存パッケージのインストール

```bash
pip install -r requirements.txt
```

### 3. Playwrightのブラウザをインストール

```bash
playwright install chromium
```

## 使い方

### アプリの起動

```bash
streamlit run app.py
```

ブラウザが自動で開きます。開かない場合は http://localhost:8501 にアクセスしてください。

### 操作方法

1. **カテゴリ選択**: サイドバーで検索したいカテゴリを選択
2. **いいね数の上限設定**: 抽出するサロンのいいね数上限を設定（デフォルト: 5）
3. **スクレイピング開始**: 「🔍 スクレイピング開始」ボタンをクリック
4. **結果確認**: 完了後、テーブルに結果が表示されます
5. **CSVダウンロード**: 「📥 CSVダウンロード」ボタンでデータをダウンロード

## 収集される情報

| 項目 | 説明 |
|------|------|
| サロン名 | サロン/スタッフ名 |
| ジャンル | ネイル、アイ（マツエク）、アイ（眉毛）、エステ・リラク |
| 住所 | サロンの住所 |
| 電話番号 | 電話番号（取得可能な場合） |
| サロンURL | ミニモのサロンページURL |
| いいね数 | いいねの数 |
| 最終更新日 | ミニモ掲載の最終更新日（詳細ページから取得） |
| 取得日時 | データを取得した日時 |

## データの保管（重要）

| 環境 | データの保存先 | 消えるタイミング |
|------|----------------|------------------|
| **Streamlit Cloud**（`*.streamlit.app`） | サーバー上の `data/salons.csv` | **アプリの再起動・デプロイ・更新のたび** |
| **ローカル**（`streamlit run`） | 同じく `data/salons.csv` | ファイルを削除したときのみ |

クラウド版では、コードを GitHub に push して再デプロイすると **昨日までのデータは消えます**（不具合ではなく仕様です）。

**必ず「CSVダウンロード」でバックアップを取ってください。** 消えた場合はアプリ左サイドバーの **「CSVから復元」** で、以前ダウンロードした `salons_*.csv` をマージできます。

保存時は次の安全ルールがあります（`tests/test_data_persistence.py` で自動テスト）:

- 空のデータで既存 CSV を上書きしない
- 行数が半分以下に急減した保存を拒否する
- 保存前に `data/backups/` へ自動バックアップ

## 注意事項

⚠️ **利用にあたっての注意**

- このツールはミニモの利用規約を遵守して使用してください
- 過度なアクセスはサーバーに負荷をかける可能性があります
- 収集したデータの取り扱いには十分注意してください
- 商用利用の際は適切な許可を得てください

## トラブルシューティング

### スクレイピングが遅い場合

- ネットワーク環境を確認してください
- `max_pages_per_category` の値を下げてみてください

### データが取得できない場合

- ミニモのサイト構造が変更された可能性があります
- Playwrightのブラウザを最新版に更新してください

```bash
playwright install chromium --with-deps
```
